<p>Justin Searls is the co-founder of Test Double, which seeks to improve how the world writes software.  He is a strong advocate for TDD, and is behind the client-side development tool <a href="http://linemanjs.com/">Lineman</a>, which "...helps developers focus on writing awesome web apps instead of worrying about workflow configuration."</p>

<p>Learn more on his <a href="http://searls.testdouble.com/">blog</a> or <a href="https://twitter.com/searls">twitter</a>.</p>
